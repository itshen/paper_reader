核心功能

为 AI 助手提供完整的论文搜索和阅读能力

智能搜索

搜索 arXiv 论文，支持智能排序、分类过滤。综合相关性和时间，让最新最相关的论文排在前面。

全文阅读

自动下载 PDF 并转换为 Markdown 格式，支持分页返回，适配 Token 限制。

智能缓存

自动缓存已下载的论文，超过 1GB 或 3 个月自动清理，节省带宽和时间。

安全认证

完整的认证机制，支持管理员登录和 API Token 管理，保护你的服务安全。

自行部署

在你自己的服务器上运行 Paper Reader MCP

1

克隆仓库

从 GitHub 获取源代码

git clone https://github.com/itshen/paper_reader.git
2

安装依赖并运行

安装 Python 依赖并启动服务

cd paper_reader
pip install -r requirements.txt
python server.py
3

配置 MCP 客户端

登录管理后台创建 Token，然后将配置添加到 Cursor 或 Claude Desktop
